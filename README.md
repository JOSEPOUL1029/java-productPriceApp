# Online Shopping Platform - Java OOP Assignment

## Overview
This project is a simple Java application that demonstrates the core principles of Object-Oriented Programming (OOP) by simulating an online shopping platform.

The application manages three product categories:
- Electronics
- Clothing
- Food Products

Each product has common attributes such as Product ID, Product Name, and Price, while each category applies its own discount policy.

## Objectives
- Implement **Abstraction** using an abstract `Product` class.
- Apply **Inheritance** for different product categories.
- Use **Encapsulation** to protect product data through private fields and getters/setters.
- Demonstrate **Polymorphism** by allowing each product type to calculate its own discount.
- Calculate the final selling price after discounts.
- Identify the most expensive product after discounts are applied.

## Features
- Add different product types
- Calculate category-specific discounts
- Display original and discounted prices
- Find the product with the highest final selling price
- Demonstrates Java OOP concepts

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile and run `Main.java`.
4. View the calculated discounts and final prices in the console.
